This 的指向问题:

普通默认函数 this 的指向 是调用者;

箭头函数的 this 的指向是父级;

箭头函数的 this 指向 settimeout 会改变 this 的指向 如果我们用箭头函数 箭头函数就指向父级。在 setInterval 和 setTimeout 中传入函数时，函数中的 this 会指向 window 对象。

This 的指向问题思考

如果箭头函数的 this 指向的是他的父级,那么我们来思考这样一个问题

  • 如果一个父级里面有一个子级,这个子集里面又嵌套了一个子集,那么在嵌套的这个最内层子集里的 this 指向是谁?
  • 如果嵌套多层,那么最内层的 this 指向是哪个父级?
