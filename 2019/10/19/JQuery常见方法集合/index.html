JQuery常见方法集合
2019-10-19
阅读量:

常用选择器

1、id 选择器

2、class 选择器

3、html 标签选择器

4、*通配符选择器

5、属性过滤选择器

6、input 表单选择器

7、过滤选择器

jQuery 的选择器和 css 的选择器有很多相似的地方，以上基本的选择器可以自由组合。比如：
$(“#id span”) //选择 id=”id”下面所有元素

$(“#id *”) //选择 id=”id”下面所有元素

常用事件

jQuery 的事件和 JS 的事件基本相同，在使用的时候，去掉 JS 事件前面的 on 即可。

  1. click：当单击鼠标时运行脚本
  2. dblclick：当双击鼠标时运行脚本
  3. mousedown：当按下鼠标按钮时运行脚本
  4. mousemove：当鼠标指针移动时运行脚本
  5. mouseout：当鼠标指针移出元素时运行脚本
  6. mouseover：当鼠标指针移至元素之上时运行脚本
  7. mouseup：当松开鼠标按钮时运行脚本

常用效果方法

  1. $(selector).hide() – 隐藏被选的元素
  2. $(selector).show() – 显示被选的元素
  3. $(selector).toggle() – 对被选元素进行隐藏和显示的切换
  4. $(selector).fadeIn() – 逐渐改变被选元素的不透明度，从隐藏到可见
  5. $(selector).fadeOut() – 逐渐改变被选元素的不透明度，从可见到隐藏
  6. $(selector).fadeTo() – 把被选元素逐渐改变至给定的不透明度
  7. $(selector).animate() – 对被选元素应用“自定义”的动画

常见的 DOM 操作方法

  1. $(selector).parent() – 返回被选元素的直接父元素。
  2. $(selector).parents() – 返回被选元素的所有祖先元素，它一路向上直到文档的根元素 ()，可选参数来过滤对祖先元素的搜索
  3. $(selector).children() – 返回被选元素的所有直接子元素。可选参数来过滤对子元素的搜索
  4. $(selector).find() – 返回被选元素的后代元素，一路向下直到最后一个后代，可选参数来过滤对后后代元素的搜索
  5. $(selector).siblings() – 返回被选元素的所有同胞元素。过滤对同胞元素的搜索。
  6. $(selector).next() – 返回被选元素的下一个同胞元素。
  7. $(selector).nextAll() – 返回匹配元素集合中每个元素之后的所有同辈元素，由选择器进行筛选（可选）。
  8. $(selector).prev() – 返回的是前面的同胞元素
  9. $(selector).prevAll() – 返回匹配元素集合中每个元素之前的所有同辈元素，由选择器进行筛选（可选）。
  10. $(selector).first() – 将匹配元素集合缩减为集合中的第一个元素。
  11. $(selector).last() – 将匹配元素集合缩减为集合中的最后一个元素。
  12. $(selector).eq() – 指定索引的新元素。
  13. $(selector).get() – 获得由选择器指定的 DOM 元素。
  14. $(selector).index() – 返回指定元素相对于其他指定元素的 index 位置。

    注：parent()、next()、prev()、first()、last()、eq()只返回一个元素。返回元素集合的函数都可以传传参数筛选，比如：parents()、siblings()、nextAll()、prevAll()、siblings()。

  15. $(selector).each() 对对象进行迭代，为每个元素执行函数
  16. $(selector).toArray() 以数组的形式返回 jQuery 选择器匹配的元素
  17. $(selector).size() 返回被 jQuery 选择器匹配的元素的数量
  18. $(selector).text() – 设置或返回所选元素的文本内容
  19. $(selector).html() – 设置或返回所选元素的内容（包括 HTML 标记）
  20. $(selector).append() – 在被选元素的结尾插入内容
  21. $(selector).prepend() – 在被选元素的开头插入内容
  22. $(selector).after() – 在被选元素之后插入内容
  23. $(selector).before() – 在被选元素之前插入内容
  24. $(selector).remove() – 删除被选元素（及其子元素）
  25. $(selector).empty() – 从被选元素中删除子元素
  26. $(selector).removeAttr() – 从所有匹配的元素中移除指定的属性。

常见的 CSS 操作

  1. $(selector).height() – 设置或返回匹配元素的高度。
  2. $(selector).width() – 设置或返回匹配元素的宽度。
  3. $(selector).addClass() – 向被选元素添加一个或多个类
  4. $(selector).removeClass() – 从被选元素删除一个或多个类
  5. $(selector).toggleClass() – 对被选元素进行添加/删除类的切换操作
  6. $(selector).css() – 设置或返回样式属性
  7. $(selector).hasClass() 检查匹配的元素是否拥有指定的类。

Ajax 操作函数

  1. $.ajax() – 执行 Ajax 异步请求
  2. $.get() – 使用 GET 请求从服务器获取数据
  3. $.getJSON() – 使用 GET 从服务器请求 JSON 编码数据
  4. $.getScript() – 使用 GET 从服务器请求 JavaScript 文件并执行该文件
  5. $.post() – 使用 POST 请求从服务器获取数据
