HTML超文本语言标签
2019-10-20
阅读量:

主体骨架部分

  • head

    • 作用: 用于存放 title,meta,style,script,link 等标签.其中必需设置的是title标签

  • title

    • 标题标签 - 给页面一个属于自己的标题.

  • body

    • 页面的主题部分,用于存放所有HTML 标签与内容的地方.

  • !DOCTYPE html

    • 因为 HTML 有很多版本,这个标签的作用就是告诉我们和浏览器当前使用的是哪个版本的 HTML,一般这个标签位于所用标签的最前面,浏览器会第一个解析我们当前文档是使用 XML 还是 HTML 标准规范

  • meta charset=”UTF-8”

    • UTF-8 是一种字符集编码方式,目前最流行并且最常用的就是 UTF-8.其他的字符集编码还有 gbk 和 gb2321 等,gb2321 是一种简单的中文编码其中有六千多个汉字

HTML 常见标签

  • 标题标签<h1~h6>
    • 为了使网页更规范更有语义化,我们常常需要在页面上使用标题标签,HTML 提供了六种标题标签也就是六个等级的标题标签,即是 h1,h2,h3,h4,h5,h6.按照等级划分 h1 是一级标签 h2 是二级标题,以此类推.一般一个页面只有一个一级标题,可以有很多个二级或三级等标题.
    • 基本语法:<h1>标题标签</h1>
  • 段落标签<p>
    • 段落标签p顾名思义,就像我们写文章一样,可以使整个页面的文字分成段落
    • 基本语法:<p>段落标签</p>
  • 水平线标签<hr />
    • 我们在浏览网站时候常常看到一些水平线将段落与段落之间隔开，使文档结构清晰，层次分明。这些水平线可以简单地通过使用水平线标签来完成,hr 就是创建水平线的标签。
    • 基本语法： <hr />是单标签
  • 换行标签<br />
    • 一个段落中的文字会从左到右按照文档流布局依次排列，直到浏览器窗口的最右端，然后自动换行。我们要实现某段文本强制换行，这时候就需要使用换行标签 br.
    • 基本语法:<br />是单标签
