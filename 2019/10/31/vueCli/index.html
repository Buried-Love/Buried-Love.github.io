VueCli安装与运行-Router-Vuex的安装
2019-10-31
阅读量:

vuecli

安装

可以使用下列任一命令安装这个新的包：

npm install -g @vue/cli
yarn global add @vue/cli

详情参考官方文档 Vue.js

步骤详解

  • 首先在需要创建 VueCli 的文件夹内黑窗口执行以下命令

    vue create 文件名

  • 选择 default 按照操作就行

  • 选择 自定义 Manually select features

  • 选择 babel; router; vuex; CSS Pre-processors (个人常用)

    • babel: ES6 语法转 ES5 语法
    • router: 路由
    • vuex: 可以理解为前端为了方便数据的操作而建立的一个 前端数据库
    • CSS Pre-processors: 减少 css3 兼容性的写法

  • node-sass 脚手架基于 node

  • 第二步勾选的插件保存到 package.json 文件里面

  • cd 刚刚创建的项目

  • npm run serve 运行

###注意事项

  • 在 main.js 引入的组件 插件 或者其他 使用 vue.use vue.component 代表注册的是全局的组件或者插件

  • 脚手架创建的项目没有页面之分 在 src 文件夹下新建 views 文件夹 这个文件夹存放的就项目全部的页面

  • 我们吧 views 下的文件夹默认看作一个页面 这是规范

  • 这个文件下下的 index.vue 就代表这个页面

  • 只有 vuex 和 vue-router 需要放到 vue 实例类里面

安装 Router

如果开始的时候没有配置 router 我们可以自行配置

  • 安装路由插件 cnpm install vue-router --save 安装不代表可以使用 要使用首先的在项目里面引入一下

  • 在 src 目录下新建文件夹 router

  • 在 router 文件夹下新建 index.js index.js 就是路由的配置文件 暴露这个配置文件 export default

  • 配置路由文件完成之后 需要在 main.js 里面进行引入 import router from "./router/index.js"

  • 把 router 配置文件挂载到 vue 实例类上面

安装 VueX

如果开始的时候没有配置 VueX 我们可以自行配置

  • 安装插件 cnpm install vuex --save

  • 在 src 文件夹下面新建 store 文件 在 store 文件下面新建 index.js

  • 配置 vuex

  • 在 main.js 里面引入 import store from "./store" 挂载到 vue 实例类上面

安装成功

  • 安装成功后可以查看版本
  • 如果有版本号证明安装正常
    执行 vue -V 命令

安装依赖

  • 我们想让脚手架跑起来还需要安装依赖

    cnpm install
