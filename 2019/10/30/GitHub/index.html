GitHub 上传

  1. gitinit 初始化包库
  2. git status 查看状态
  3. git add . 上传到暂存区
  4. git commit -m “提交信息”
  5. git pull 拉去当前分支最新代码
  6. git push origin master | push 到远程 master 上

然后把刚才获取到的源代码的链接复制过来，git clone 链接。
